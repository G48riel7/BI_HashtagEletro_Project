# BI_HashtagEletro_Project

# 📊 Dashboard de Vendas — Hashtag Eletro

> Análise de desempenho comercial de uma rede de varejo de eletrônicos utilizando Power BI.

---

## 📌 Sobre o projeto

Este projeto consiste em um dashboard interativo desenvolvido em **Power BI** para análise de vendas de uma rede fictícia de varejo de eletrônicos chamada **Hashtag Eletro**.

O objetivo foi simular um cenário real de Business Intelligence, onde o time de gestão precisa acompanhar KPIs de vendas, identificar tendências e apoiar decisões estratégicas com base em dados.

---

## 🎯 Problema de negócio

> Como o time comercial pode identificar rapidamente quais produtos, categorias e períodos estão performando abaixo do esperado, e onde concentrar esforços para recuperação de resultados?

---

## 💡 Solução desenvolvida

Dashboard com visões integradas que permitem ao gestor:

- Monitorar **faturamento total, ticket médio e volume de vendas**
- Comparar desempenho por **categoria de produto e período**
- Identificar os **produtos mais e menos vendidos**
- Acompanhar a **evolução mensal das vendas**
- Filtrar por período, categoria e loja de forma dinâmica

---

## 📈 Principais métricas do dashboard

- **Faturamento Total**
- **Ticket Médio por Venda**
- **Volume de Unidades Vendidas**
- **Top 5 Produtos por Receita**
- **Evolução Mensal de Vendas**
- **Participação por Categoria**
